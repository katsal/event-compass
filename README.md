## Getting Started

#### Setup

Install gems

`bundle install
`
#### ENV Variables

Create `.env` file

`touch .env`

Inside `.env`, set these variables. 

`CLOUDINARY_URL=your_own_cloudinary_url_key`

#### DB Setup

```
rails db:create
rails db:migrate
rails db:seed
```
Run a server

`rails s
`
